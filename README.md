AI Risk & Financial Services Impact Analysis

A portfolio project exploring how AI adoption may affect financial services through operational efficiency, fraud risk, workforce productivity, and governance. This project is designed to support applications to AI, fintech, finance strategy, and research-focused roles such as Anthropic's Economics & Societal Impacts or finance-related opportunities.

## Project Overview

Advanced AI systems are increasingly being used in finance for customer support, fraud detection, credit operations, reconciliation, reporting, forecasting, and process automation. This project analyses the benefits and risks of AI adoption in financial services and proposes a lightweight risk scoring framework for responsible deployment.

## Objectives

- Analyse potential AI use cases in financial services.
- Identify key operational, financial, compliance, and societal risks.
- Build a simple AI risk scoring model using Python.
- Create a structured framework for evaluating whether an AI use case is suitable for deployment.
- Present findings in a clear, finance-and-strategy-oriented format.

## Why This Project Matters

AI adoption in finance can improve speed, accuracy, and scalability, but it can also introduce new risks around bias, explainability, data privacy, over-reliance, model failure, and governance. Responsible deployment requires both technical understanding and financial control discipline.

## Methodology

The project uses a sample dataset of AI use cases in finance and scores each case across several risk dimensions:

- Financial impact
- Operational complexity
- Regulatory sensitivity
- Data privacy exposure
- Explainability requirement
- Human oversight requirement

Each factor is scored from 1 to 5. A weighted risk score is then calculated and categorised as Low, Medium, or High.

## How to Run

1. Clone this repository.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the risk scoring script:

```bash
python src/risk_scoring.py
```

4. Open the notebook in Jupyter for analysis and visualisation:

```bash
jupyter notebook notebooks/ai_risk_analysis.ipynb
```
